# Ingama_Invoice

Standalone invoice form for Ingama Textil, prepared as a static site.

## English

### What is included

- `index.html`: main invoice page
- `styles.css`: interface styles and print layout
- `assets/logo.jpeg`: logo used in the document

### How to use it

1. Open `index.html` in your browser.
2. Fill in the client and issuer details.
3. Add or remove invoice lines as needed.
4. Adjust quantity, unit price, and VAT.
5. Use **Download PNG** to save the invoice as a PNG image. If the browser cannot convert it, the page falls back to SVG automatically.
6. Use **Print / PDF** to print it or save it as a PDF from the browser.
7. Use **Reset** to clear the form and start again.

### GitHub Pages

This repository is ready to be published directly from the root because `index.html` is in `/`.

## Ukrainska

### Що є в репозиторії

- `index.html`: основна сторінка рахунку
- `styles.css`: стилі інтерфейсу та версії для друку
- `assets/logo.jpeg`: логотип, який використовується в документі

### Як користуватися

1. Відкрий `index.html` у браузері.
2. Заповни дані клієнта та постачальника.
3. Додай або видали рядки рахунку за потреби.
4. Зміни кількість, ціну за одиницю та ставку ПДВ.
5. Натисни **Download PNG**, щоб зберегти рахунок як PNG-зображення. Якщо браузер не зможе виконати перетворення, сторінка автоматично завантажить SVG.
6. Натисни **Print / PDF**, щоб роздрукувати або зберегти документ як PDF із браузера.
7. Використовуй **Limpiar campos / Reset**, щоб очистити форму і почати знову.
