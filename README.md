# Ingama_Invoice

Formulario de factura autonomo para Ingama Textil, preparado como sitio estatico.

## Contenido

- `index.html`: pagina principal del formulario
- `styles.css`: estilos de la interfaz y version imprimible
- `assets/logo.jpeg`: logo usado en el documento

## Uso local

Abre `index.html` en el navegador.

## Publicar con GitHub Pages

1. Ve a **Settings > Pages** en el repositorio.
2. En **Build and deployment**, elige **Deploy from a branch**.
3. Selecciona la rama **main** y la carpeta **/(root)**.
4. Guarda los cambios.

La web quedara publicada desde la raiz del repositorio porque `index.html` ya esta en `/`.
